# LaTeX Figures

- `\includegraphics[width=\linewidth]{...}` — scale to column width
- `\centering` inside figure env (not `\begin{center}`)
- Caption BELOW figure, ABOVE table (convention)
- Subfigures: `subcaption` package, `\begin{subfigure}[b]{0.48\linewidth}`
- PDF > PNG > JPG for vector vs raster
