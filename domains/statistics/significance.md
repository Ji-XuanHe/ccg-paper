# Statistical Significance

- Report mean ± std over ≥3 seeds (5+ preferred)
- Use paired test when comparing methods on same data (paired t / Wilcoxon)
- Correct for multiple comparisons (Bonferroni / Holm) when N_tests ≥ 5
- State the test used ("paired two-sided t-test, α=0.05")
- p-value alone is not enough — report effect size too
