# ICML — Reviewer Preferences

- **Format**: 8 pages main + 2 pages refs + appendix. Double blind. `icml20XX.sty`.
- **Review model**: AC-driven; 3 reviewers; rebuttal tight.
- **Preference**: Theoretical grounding + clean empirical. "Principled" over "large-scale hacks".
- **Style**: Mathier than NeurIPS on average; proofs in appendix OK but main should motivate.
- **Common failure**: weak theoretical backing for empirical trick.
- **Ethics/Repro**: Encouraged, not always enforced; code release expected for empirical papers.

## Watch out
- AC can desk-reject if no baseline comparison present
- Overly long related work penalized
