# ACL / EMNLP / NAACL — Reviewer Preferences

- **Format**: 8 pages (long) / 4 pages (short) + unlimited refs + limitations (required).
- **ARR rolling review**: Submit to ACL Rolling Review, then commit to a venue.
- **Required sections**: Limitations + Broader Impact / Ethics (mandatory, not optional).
- **Reproducibility**: Detailed hyperparam tables, link to code/data.
- **Responsible NLP Checklist**: Submit during review.

## Preferences
- Linguistically-grounded analysis valued over pure leaderboard chasing
- Error analysis expected for any NLP paper
- Multilingual / low-resource angles get bonus

## Common rejects
- Only English benchmarks without justification
- No human evaluation for generation tasks
- Prompt engineering without ablation
