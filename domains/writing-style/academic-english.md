# Academic English — Core Rules

- **Voice**: Mix active + passive. "We propose" (active) for claims; "is evaluated on" (passive) for methods.
- **Hedging**: Use "suggests / indicates / can improve" for empirical claims; "proves / guarantees" only for theorems.
- **Banned**: "obviously", "clearly", "simply", "of course" (assumes reader).
- **Reduce**: "in order to"→"to", "due to the fact that"→"because", "a large number of"→"many".
- **First person**: "we" is standard (even single-author). Avoid "I" unless journal style allows.
