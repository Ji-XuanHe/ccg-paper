# Colormap Choice

- **Sequential**: viridis (default), plasma, magma — perceptually uniform
- **Diverging**: coolwarm, RdBu — for signed data around 0
- **Qualitative**: tab10, Set2 — for categories
- **NEVER**: jet / rainbow (not uniform; bad for colorblind)
- **Colorblind check**: Simulate with coblis or Color Oracle
