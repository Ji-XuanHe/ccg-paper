# Rebuttal Structure

Standard 1-page layout:
1. **Summary of response** (2-3 lines) — what you changed, key new results
2. **Per-reviewer sections** — R1 / R2 / R3
3. **Common concerns** (if 2+ reviewers raised) — dedup here
4. **New experiments** (if any) — highlight with table
5. **Commitments** — what you'll add in camera-ready

Group by reviewer if concerns differ; by theme if overlap.
